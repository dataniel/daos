# Split a data frame into a named list by grouping columns

Groups a data frame and splits it, automatically naming each element of
the resulting list from the unique group key values. Unlike the bare
[`dplyr::group_split()`](https://dplyr.tidyverse.org/reference/group_split.html),
the returned list is always named.

## Usage

``` r
split_by(data, ..., .sep = "_")
```

## Arguments

- data:

  A data frame or tibble.

- ...:

  Grouping columns (unquoted, tidy-select).

- .sep:

  Separator used to concatenate multiple group key values into a single
  list name. Default: `"_"`.

## Value

A named list of tibbles.

## See also

[`summon()`](https://dataniel.github.io/daos/reference/summon.md) to
retrieve objects matching a name pattern
